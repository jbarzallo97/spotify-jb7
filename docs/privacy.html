Política de Privacidad – spotify-jb7

Responsable: Johan Barzallo · johanbarzallo11@gmail.com

Qué datos accedemos

La app utiliza la Spotify Web API y únicamente accede a los datos que tú autorizas expresamente al iniciar sesión. Los permisos solicitados son:

user-read-private, user-read-email, user-follow-read, user-top-read, user-read-recently-played, playlist-read-private, playlist-read-collaborative.

Para qué usamos tus datos

Exclusivamente para mostrar en la interfaz tu perfil, artistas y canciones más escuchados, recientes y playlists. No hay otras finalidades.

Tratamiento y conservación

  • No existe servidor propio que almacene tu información; todo se procesa en tu navegador.
  • El token de acceso de Spotify se guarda en el almacenamiento del navegador (localStorage) para mantener la sesión y expira según las políticas de Spotify.
  • Al cerrar sesión se eliminan los datos de la sesión del navegador.

Compartición con terceros

No vendemos ni compartimos tus datos con terceros. Solo interactuamos con el servicio oficial de Spotify para leer la información autorizada.

Base legal

El tratamiento se basa en tu consentimiento otorgado mediante el flujo OAuth de Spotify. Puedes retirarlo en cualquier momento.

Revocar acceso

Puedes revocar el permiso desde tu cuenta de Spotify (Aplicaciones) o usando el botón de “Logout” de la app. Tras revocar, la app ya no podrá acceder a tus datos.

Seguridad

Se aplican prácticas estándar (uso de HTTPS de GitHub Pages y tokens temporales) para proteger la comunicación con Spotify.

Menores

El servicio está dirigido a mayores de 13 años. Si detectas acceso indebido, contáctanos para retirar permisos.

Cambios a esta política

Podremos actualizar esta página para reflejar mejoras o cambios normativos. La versión vigente será la publicada aquí.
